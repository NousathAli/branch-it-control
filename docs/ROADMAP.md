# Roadmap

## Foundation
- Split the monolithic application into modules.
- Add automated tests and validation.
- Implement structured master data: Category -> Asset Type -> Brand -> Model.

## Security
- Migrate to Firebase Authentication or Microsoft Entra ID.
- Add role claims and Firestore Security Rules.
- Move vault encryption to a protected backend.
- Add audit logs and MFA.

## Asset lifecycle
- Asset movement, repair, warranty, disposal, attachments, and parent-child relationships.
- Camera-to-NVR and HDD-to-NVR mapping.

## Reporting
- Better PDF/Excel reports, dashboards, compliance reports, QR labels, and preventive maintenance.
