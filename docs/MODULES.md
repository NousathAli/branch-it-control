# Modules

## Dashboard
Project, location, branch, and asset summaries with status and category charts.

## Groups and Brands
Business group, concept, and branding management.

## Locations and Branches
Location master, branch projects, operational details, and branch-level navigation.

## IT Assets
Global asset register, reusable asset products, category master, standard branch templates, and Excel workflows.

## CCTV/NVR
Camera deployment summaries, operation filters, NVR recording-day review, and weekly review tracking.

## Credential Vault
Shared accounts, device credentials, and user access assignments with browser-side encryption for the prototype.

## Fault Logs
Operational issue recording, status tracking, and report export.

## Handover and Pullout
Department handover and asset pullout documentation with printable records.

## User Portfolio
Professional profile, project involvement, fault contribution, handovers, pullouts, and CCTV review evidence.

## Settings
Theme, users and access, password-reset requests, category master, database backup, and related controls.
